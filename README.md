# Marketing Campaign Performance & A/B Testing Analysis
## Overview
This project analyzes a marketing campaign dataset to evaluate performance, identify issues, and provide data-driven recommendations to improve conversion and retention.The analysis focuses on understanding which marketing channels perform best, how different customer segments behave, and how A/B testing can be used to optimize decision-making.

## Business Problem
A marketing team wants to:
- Evaluate campaign performance across channels
- Understand factors influencing conversion rates
- Identify and resolve performance issues
- Use A/B testing to improve campaign effectiveness

## Tools & Technologies
- Python (pandas, NumPy)
- Matplotlib (visualization)
- Jupyter Notebook

## Data Exploration & Preprocessing
- Inspected dataset using .head(), .info(), and .describe()
- Identified and handled data inconsistencies
- Performed feature engineering by creating new columns
- Cleaned and prepared data for accurate analysis

## Key Analysis
## 1. Marketing Performance Metrics
- Calculated conversion rate and retention rate
- Compared performance across different marketing channels
## 2. Customer Segmentation
- Analyzed user behavior across:
- Marketing channels
- Age groups
- Regions
- Identified differences in engagement and conversion
## 3. Conversion Drop Investigation
- Detected a significant decline in conversion rates
- Investigated root causes using segmented analysis
- Identified a language mismatch issue affecting specific user groups
## 4. A/B Testing Analysis
- Compared performance between test variants
- Calculated lift to measure improvement
- Performed statistical significance testing
- Evaluated results across different segments
# Key Insights
- High traffic does not always translate to high conversion
- Certain channels consistently outperform others in efficiency
- Data quality issues (e.g., language mismatch) can significantly impact results
- A/B test results can vary across user segments
